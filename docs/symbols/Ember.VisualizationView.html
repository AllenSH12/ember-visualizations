Class: Ember.VisualizationView

VisualizationView provides a base visualization class for rendering graph axes, performing axis and label formatting, and displaying gridlines.

In any of your views, simply override xFormatter, and yFormatter to provide custom axis formatting.

VisualizationView is also responsible for redrawing the element on the page when the content changes.
